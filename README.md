# TRM-Geometry-Core
Fundamentalna geometria skrętu i struktur w TRM

## Cel
Repozytorium definiuje podstawowe figury i przejścia geometryczne:
- koło (cykl skrętu)
- trójkąt równoboczny wpisany
- kwadratura koła (fala → struktura)
- kula (pełne pole)
- sześcian wpisany w kulę (materia stabilna)

## Struktury
### Koło
S = π r²

### Trójkąt wpisany
a = r√3

### Kwadratura koła
a = r√π

### Kula
V = 4/3 π r³

### Sześcian wpisany w kulę
a = 2r / √3
