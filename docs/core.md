# CORE — Aksjomaty geometryczne TRM

Poniższy zestaw stanowi minimalny, zamknięty zbiór zależności
opisujących przejścia geometryczne w TRM-Geometry-Core.

Każda figura i każdy operator ma dowód matematyczny
i jest spójny z pozostałymi elementami.

-----------------------------------------
1. KOŁO — cykl skrętu
-----------------------------------------
Pole:
S = π r²

Aksjomat:
Koło jest podstawową figurą cyklu.
Promień r jest jednostką skali.

-----------------------------------------
2. TRÓJKĄT RÓWNOBOCZNY — trzy stany
-----------------------------------------
Bok trójkąta wpisanego w koło:
a = r√3

Aksjomat:
Trójkąt reprezentuje trzy stabilne stany.
Operator przejścia: √3

-----------------------------------------
3. KWADRATURA KOŁA — fala → struktura
-----------------------------------------
Warunek równoważności pól:
a² = π r²

Bok kwadratu:
a = r√π

Aksjomat:
Kwadrat jest strukturą równoważną kołu.
Operator przejścia: √π

-----------------------------------------
4. KULA — pełne pole skrętu
-----------------------------------------
Objętość:
V = (4/3) π r³

Aksjomat:
Kula jest trójwymiarowym odpowiednikiem koła.
Promień r zachowuje skalę.

-----------------------------------------
5. SZEŚCIAN — stabilna materia w polu
-----------------------------------------
Przekątna przestrzenna:
d = a√3

Warunek wpisania w kulę:
d = 2r

Bok sześcianu:
a = 2r / √3

Aksjomat:
Sześcian jest stabilną strukturą w polu kuli.
Operator przejścia: 2/√3

-----------------------------------------
6. SPÓJNOŚĆ OPERATORÓW
-----------------------------------------
Trójkąt ↔ Sześcian:
  operator: √3

Koło ↔ Kwadrat:
  operator: √π

Kula ↔ Sześcian:
  operator: 2/√3

Aksjomat:
Każde przejście między figurami jest
transformacją skali bez utraty informacji.

-----------------------------------------
7. ZAMKNIĘCIE CORE
-----------------------------------------
Zbiór { koło, trójkąt, kwadrat, kula, sześcian }
jest zamknięty względem operatorów:
{ √3, √π, 2/√3 }

To stanowi pełny rdzeń geometryczny TRM.
