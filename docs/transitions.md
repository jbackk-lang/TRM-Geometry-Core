# Przejścia między skalami — operator skrętu

Przejście między figurami opisuje zmianę poziomu:
- koło → trójkąt → kwadrat → kula → sześcian

Każde przejście ma operator:

1. Koło → Trójkąt
   operator: √3
   (trzy stany stabilne)

2. Koło → Kwadrat
   operator: √π
   (fala → struktura)

3. Koło → Kula
   operator: r
   (rzut 2D → 3D)

4. Kula → Sześcian
   operator: 2/√3
   (materia w polu)

Interpretacja:
Przejście między skalami to zmiana operatora skrętu,
bez utraty informacji i bez zmiany fazy.
