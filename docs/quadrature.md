# Kwadratura koła — przejście fala → struktura

Kwadratura koła opisuje przejście:
cykl (koło) → siatka (kwadrat)

Warunek równoważności pól:
a² = π r²

Bok kwadratu:
a = r√π

Interpretacja:
- przejście z fali do struktury
- operator √π jako transformacja poziomu
- fundament przejścia między skalami
