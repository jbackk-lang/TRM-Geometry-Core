# Matematyczne dowody zależności geometrycznych w TRM-Geometry-Core

## 1. Trójkąt równoboczny wpisany w koło
Niech r będzie promieniem okręgu opisanego na trójkącie równobocznym o boku a.

Wysokość trójkąta:
h = (√3 / 2) a

Promień okręgu opisanego:
R = (2/3) h = (2/3)(√3/2)a = (√3/3)a

Warunek:
R = r

Zatem:
r = (√3/3)a
a = r√3
## 2. Kwadratura koła — równoważność pól
Pole koła o promieniu r:
S₁ = π r²

Pole kwadratu o boku a:
S₂ = a²

Warunek kwadratury:
a² = π r²

Po spierwiastkowaniu:
a = r√π
## 3. Objętość kuli
Rozpatrz przekrój kuli na wysokości z.

Promień przekroju:
ρ(z) = √(r² - z²)

Pole przekroju:
S(z) = π (r² - z²)

Objętość:
V = ∫ od -r do r π (r² - z²) dz
  = π [ r²z - z³/3 ] od -r do r
  = π ( (r³ - r³/3) - (-r³ + r³/3) )
  = π ( 2r³ - 2r³/3 )
  = (4/3) π r³
## 4. Sześcian wpisany w kulę
Niech a będzie bokiem sześcianu wpisanego w kulę o promieniu r.

Przekątna przestrzenna sześcianu:
d = a√3

Promień kuli opisanej:
R = d/2 = (a√3)/2

Warunek:
R = r

Zatem:
r = (a√3)/2
a = 2r / √3
## 5. Spójność operatorów przejścia
Zależności:

Trójkąt:
a = r√3

Kwadrat:
a = r√π

Sześcian:
a = 2r / √3

Wnioski:
- √3 łączy trójkąt i sześcian (obie figury związane z trójką).
- √π łączy koło i kwadrat (fala → struktura).
- 2/√3 łączy kulę i sześcian (średnica → przekątna).
